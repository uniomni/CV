Examples of how fortran(77) and Python can be linked.


You will need to install py2f (available at GA in Downloads/Python) on your windows machines but it is already installed on the Linux box nautilus.

See also http://www.mingw.org/MinGWiki/index.php/Python%20extensions for info on using MinGW and Python. It helped me.
  

Run the tests

test_interpol.py and
array_test.py  
  

and look at the fortran files
