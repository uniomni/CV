This is a Python implementation of the wavelet transforms described in
my thesis

  http://datamining.anu.edu.au/~ole/work/publications/thesis.html
  
The older Matlab implementation is available at   

  http://datamining.anu.edu.au/~ole/work/software/wavpak
  
  
  
To install and test, run

python test_daubfilt.py
python test_wavelets.py


Requirements:

* Python
* Numerical Python
* A C compiler such as gcc (MinGW for Windows)
